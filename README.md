# Haskell

Haskell is pure functional programming language
- Pure functions
- Extremely lazy
- Type inference

## Running haskell
Download ghci and then:
```bash
ghci

Prelude>
```

Or edit a file with `*.hs` extension and then:
```bash
runhaskell file.hs
```
